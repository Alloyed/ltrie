Module List

A naive singly linked list, similar to lisp lists.

Like Vector, they are dense, ordered, begin at one, and (at least in this implementation) are immutable.

Class List

List:len ()
Returns the number of values contained by the list. In lua 5.2 and up, #list can be used as a shortcut.

complexity:

    O(n)

Usage:

    List.of(1, 2, 3):len() == 3
List:ipairs ()
Iterate through the list, from beginning to end.

Note that instead of returning index, value per iteration like normal ipairs(), the value of index is implementation-defined.

Usage:

    for _it, v in my_list:ipairs() do print(v) end
List:conj (val)
Returns a new list with the value added to the beginning.

complexity:

    O(1)

Parameters:

  • val the value to add

Usage:

    List.of(2, 3):conj(1) == List.of(1, 2, 3)
List:pop ()
Returns a new list with the first value removed.

complexity:

    O(1)

Usage:

    List.of(1, 2, 3):pop() == List.of(2, 3)
List:assoc (idx, val)
Returns a new list with the value at index idx replaced with val.

complexity:

    O(n)

Parameters:

  • idx int the index
  • val the value

Usage:

    List.of(0, 0, 0):assoc(2, 'a') == List.of(0, 'a', 0)
List:get (idx)
Returns the value at index idx.

complexity:

    O(n)

Parameters:

  • idx int the index

Returns:

Usage:
