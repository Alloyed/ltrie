Module Subvec

A subsequence of a vector.

Subvecs can only be constructing from an existing, non-transient vector or subvec, and otherwise support everything a normal vector would.

Functions

new (vec, vecStart, vecEnd) Creates a new subvector s containing the elements from vec within the range [vecStart, vecEnd] such that s:get(1) == vec:get(vecStart) and s:get(s:len()) == vec:get(vecEnd)
len () Returns the number of elements in the subvector.
get (idx) Get a value by index.

conj (val) Returns a new subvector with val appended to the end.

Functions

new (vec, vecStart, vecEnd)
Creates a new subvector s containing the elements from vec within the range [vecStart, vecEnd] such that s:get(1) == vec:get(vecStart) and s:get(s:len()) == vec:get(vecEnd)

Parameters:

  • vec the vector
  • vecStart the start index of the subvector, inclusive
  • vecEnd the end index of the vector, inclusive

Usage:

    sv = Subvec.new(Vector.of(1, 2, 3, 4), 2, 4) -- [2, 3, 4]
len ()
Returns the number of elements in the subvector. In lua 5.2 and up, #subvec can be used as a shortcut.

complexity:

    O(1)

Usage:

    sv:len() == 3
get (idx)
Get a value by index. Vector indexes start at one.

complexity:

    O(log32 n)

Parameters:

  • idx int the index

Returns:

    the value, or nil if not found

Usage:

    sv:get(2) == 3
assoc (idx, val)
Returns a new subvector such that sv:get(idx) == val. This creates a new parent vector to represent the modification.

complexity:

    O(log32 n)

Parameters:

  • idx int the index to set
  • val the value to set

Usage:

    Vector.of(0, 0, 0):assoc(2, 'a') == Vector.of(0, 'a', 0)
conj (val)
Returns a new subvector with val appended to the end. This creates a new parent vector to represent the modification.

complexity:

    O(log32 n)

Parameters:

  • val the value to append

Usage:

    sv:conj(5) == Subvec.new(Vector.of(1, 2, 3, 4, 5), 2, 5)
pop ()
Returns a new vector with the last value removed.

complexity:

    O(1)

Usage:

    Vector.of(1, 2, 3):pop() == Vector.of(1, 2)
